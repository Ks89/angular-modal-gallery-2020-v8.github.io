Global configuration

If you want to customize a particular behaviour of this library for the entire application (GLOBAL CONFIGURATION), you can pass an object to the forRoot() method in your root module.

Attention: you can only add a behaviour to this library across your entire application.


How to use it?

Into your root module, pass an object of type KeyboardServiceConfig to the forRoot() method.

For example, to download images also with ctrl+a use this:


  @NgModule({
    declarations: [AppComponent],
    imports: [
      BrowserModule,
      GalleryModule.forRoot( { shortcuts: ['ctrl+a', 'ctrl+s', 'meta+s'] } )
    ],
    bootstrap: [AppComponent],
  })
  export class AppModule {}

KeyboardServiceConfig

Interface to change global download shortcuts.


shortcuts
Type: string[] | string

This is a direct input into KeyboardService (mousetrap.bind(HERE, ...)) to manage keyboard's shortcuts. Please refer to the official documentation of mousetrap. By default this value is ['ctrl+s','meta+s'].
