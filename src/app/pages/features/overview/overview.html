Overview

@ks89/angular-modal-gallery is composed mainly in three parts: Carousel, Plain gallery and Modal gallery. However, these components share some common interfaces and classes. So, I splitted this section in different parts: basic stuff and main components.


Basic stuff

  • Common: all shared interfaces and classes
  • Description: interfaces to customize description for both Modal gallery and Carousel
  • Accessibility: interface to change accessibility texts for all components
  • Previews: interfaces to configure previews for both Modal gallery and Carousel
  • UpperButtons: interface to customize buttons of Modal gallery
  • Global configuration: explain how to change the behaviour of the library for the entire application passing parameters to forRoot() method
  • Default values: default values for all inputs of all components

Components

  • Carousel: available via <ks-carousel></ks-carousel>
  • Modal gallery: available via <ks-modal-gallery></ks-modal-gallery>
  • Plain gallery: included inside Modal gallery, so it's not available via a standalone tag, but you have to use <ks-modal-gallery></ks-modal-gallery> passing the right inputs

Carousel


It contains a list of scrollable images (full width or fixed width) also with previews. By default images scroll automatically. Users can configure this component with many parameters, also they can project HTML content into the current carousel image.

For more info check Carousel's documentation


Modal gallery


  1. Current image
  2. Side previews
  3. Upper buttons
  4. Dots
  5. Previews
  6. Background (semi-transparent)
  7. Modal gallery (the main component)

For more info check Modal gallery's documentation


Plain gallery


It contains a list of thumbnails that users can click to open modal gallery. Layouts are configurable and users could also use a custom implementation to draw everything they want.

For more info check Plain gallery's documentation
